URLs: Addresses on the Web

A Uniform Resource Locator (URL) is the address of a specific resource on the Internet, like a webpage or an image. It tells your browser where to find what you're looking for.

Interactive URL Breakdown:

https://www.example.com/products/category/item.html
Click a part of the URL above!

Learn about the different components that make up a web address.

Scheme (e.g., `https://`)

Indicates the protocol used to access the resource. `https` (Hypertext Transfer Protocol Secure) is the most common and secure protocol for web pages.

Subdomain (e.g., `www.`)

An optional part that specifies a subdivision of the domain. `www` (World Wide Web) is common but not always present.

Domain Name (e.g., `example`)

The unique name that identifies a website or organization on the internet.

Top-Level Domain (TLD) (e.g., `.com`)

The last segment of the domain name, indicating the type or country of the organization (e.g., `.com` for commercial, `.org` for organization, `.gov` for government, `.uk` for United Kingdom).

Path (e.g., `/products/category/`)

Specifies the exact location of a page or resource within the website's structure, similar to folders on your computer.

File Name (e.g., `item.html`)

The specific file being requested on the server. This might be a webpage, an image, a document, or other resource.
