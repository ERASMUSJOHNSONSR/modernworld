MordernWorld Moving and Copying Files

Learn how to organize your digital content by moving files to new locations or creating duplicates. This guide covers both drag & drop and copy & paste methods.

Moving Files: Relocating Content

Moving a file means relocating it from its original spot to a new destination. The file will no longer exist in the first location. The most common way to move files is using drag & drop.

Ready to move a file?

Click "Simulate Move" to see drag & drop in action.

Step 1: Locate File

Find the file you want to move in its current folder.

Step 2: Drag & Drop

Click and hold the file with your mouse. Drag it to the new folder (destination) and release the mouse button.

Result: File Moved

The file is now in the new folder and no longer in its original location.

Copying Files: Creating Duplicates

Copying a file creates an exact duplicate, leaving the original file untouched in its location. This is useful for backups or working with different versions.

Ready to copy a file?

Click "Simulate Copy" to see the process using shortcuts.

Step 1: Select & Copy

Click the file you want to copy. Press Ctrl + C (Windows) or Cmd + C (Mac) to copy it.

Step 2: Navigate & Paste

Go to the destination folder. Press Ctrl + V (Windows) or Cmd + V (Mac) to paste the copy.

Result: File Duplicated

The file now exists in both the original and new locations.

Copying with Drag & Drop (Advanced)

You can also copy files using drag & drop, but you need to hold down a modifier key:

  • **Windows:** Hold down the Ctrl key while dragging the file to its new location.
  • **macOS:** Hold down the Option key while dragging the file to its new location.

This method creates a copy, leaving the original file intact, just like Copy & Paste.
